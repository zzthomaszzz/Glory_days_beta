# GloryDay Beta

A fast-paced **2v2 multiplayer top-down action game** built with Python and Pygame.

Pick a hero, fight for map control, destroy the enemy HQ, and win.

---

## Heroes

| Hero | Playstyle |
|------|-----------|
| **Soldier** | Ranged marksman — Snipe, turret placement, dash |
| **Mage** | Burst caster — Fireball, heal, blink with shockwave |
| **Hunter** | Tanky brawler — Charge, ground slam, fortify |
| **Samurai** | Melee duelist — Spin, Bushido strike, rally banner |

All heroes share a **Recall** ability (B) — channel 4 seconds to teleport back to base.

---

## How to Play

1. Download `GloryDay_v1.0.zip` from [Releases](https://github.com/zzthomaszzz/Glory_days_beta/releases)
2. Extract the zip
3. Run `main.exe`
4. Enter the server IP and connect

**One player hosts the server, others connect to their IP.**

---

## Controls

| Input | Action |
|-------|--------|
| Left click | Move / Attack |
| Right click | Sell item (on inventory slot) |
| Q / E / R | Hero abilities |
| B | Recall to base |
| F | Open shop (near HQ) |
| Space | Snap camera to hero |
| C | Toggle attack range |

---

## Objectives

- **Kill enemies** to earn gold
- **Buy items** at your HQ shop to power up
- **Destroy the enemy HQ** to win

---

## Requirements

- Windows
- No install needed — just extract and run `main.exe`
